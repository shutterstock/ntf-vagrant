# ntf Vagrant

This is an ntf development environment for Vagrant.

## Requirements

  * [Node.js](http://nodejs.org/)
  * [Vagrant](http://vagrantup.com/)

## Getting Started

Setup environment

    make build

Run server

    make server

Run daemon

    make daemon

Open [localhost:8000](http://localhost:8000/)
